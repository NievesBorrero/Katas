# Katas

## Gilded Rose kata

[Specifications](https://kata-log.rocks/gilded-rose-kata)

### Getting started

Install dependencies

```sh
npm install
```

### Running tests

To run all tests

```sh
npm test
```

To run all tests in watch mode

```sh
npm run test:watch
```

To generate test coverage report

```sh
npm run test:coverage
```
